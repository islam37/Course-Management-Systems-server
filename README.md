# Course Management System - Client

A **responsive web application** for browsing, enrolling, and managing courses.  
This project demonstrates practical skills in React.js, TailwindCSS, Firebase Authentication, and integration with a Node.js/Express backend with MongoDB.

## 🌐 Live Demo
[View Live Site](https://course-management-system-5c841.web.app/)

## ✨ Key Features
- User registration & login with **Firebase Auth** (email/password & social login)
- Browse latest & popular courses
- View course details with images and descriptions
- **Enroll/Unenroll in courses** with seat limitation (max 3 courses per user)
- Add, edit, and delete courses (only for logged-in users)
- Dynamic page titles and toast notifications
- Responsive design for **mobile, tablet, and desktop**
- Smooth animations using **Framer Motion**
- Slider/hero section using **react-slick**

## 🛠️ Tech Stack
- **Frontend:** React.js, TailwindCSS, Mantine UI, Axios, react-router-dom
- **Backend API:** Node.js, Express.js, MongoDB
- **Authentication:** Firebase Authentication
- **Other Tools:** react-slick (carousel), framer-motion (animations), toast notifications
